# dotfiles

My config files

## Install

`git clone git@github.com:boldyrev/dotfiles.git ~/dotfiles`

`sudo pacman -S tmux lazygit lazydocker ttf-firacode-nerd opencode yay node pnpm docker docker-compose neovim`

`yay -S asdf-vm`

`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`

`ln -s ~/dotfiles/.tmux.conf ~/`

Run tmux and and press `<C-f I>` at first run for install plugins

After install nvim, run `:LazyExtras` and check Elixir and other extras
