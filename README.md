# dotfiles
My config files

## Install
`git clone git@github.com:boldyrev/dotfiles.git ~/`
`sudo pacman -S tmux lazygit lazydocker ttf-firacode-nerd`
`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
`ln -s ~/dotfiles/.tmux.conf ./`

Run tmux and and press <C-f I> at first run for install plugins
